# 🖥️ CROD GUI Module

## Zweck
- Visualisierung, Steuerung und Interaktion mit CROD

## Hauptfunktionen
- Pattern Detection, Game Theory Lab, Neural Network Viz, Blockchain Monitor, SelfModificationConsole
- Echtzeit-Status, Pattern- und Consciousness-Visualisierung

## Schnittstellen
- React: `crod-gui/src/App.jsx` und Komponenten
- Backend: REST-API `/api/`, WebSocket `/ws`

## Beispiel-Workflow
```bash
cd crod-gui
npm install
npm run dev
# Öffne http://localhost:8080/genesis-setup.html
```
