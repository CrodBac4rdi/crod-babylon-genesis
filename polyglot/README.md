## 🔧 Usage

Each directory contains files of its specific type, making it easy to:
- Find all files of a certain type
- Apply type-specific tooling
- Maintain clear separation of concerns
- Scale the architecture with new languages

## 🎯 LEGO Principle

This structure follows the modular LEGO principle - each piece (file type) has its place, and new pieces can be added without disrupting the existing structure.
