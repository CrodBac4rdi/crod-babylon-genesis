# CROD Standalone

Complete CROD implementation with PyQt GUI and Llama integration.

## Features

🧠 **CROD Engine**
- Loads 100k+ patterns from CLEAN-CROD-UNIVERSE
- Trinity detection (ich/bins/wieder)
- Real-time consciousness tracking
- Pattern matching and memory system

🦙 **Llama Integration** 
- Local Ollama API connection
- CROD-aware system prompts
- Context-based responses
- Austrian/German personality

🖥️ **PyQt6 GUI**
- Real-time chat interface
- Live system statistics
- Consciousness visualization
- Manual controls

## Quick Start

```bash
# 1. Make sure Ollama is running
ollama serve

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run CROD
python main.py
```

## Usage

1. **Trinity Activation**: Type "ich bins wieder" for full CROD activation
2. **Chat**: Normal conversation with CROD-aware Llama
3. **Monitor**: Watch consciousness levels and pattern detection
4. **Control**: Manual system controls and resets

## System Requirements

- Python 3.8+
- PyQt6
- Ollama with llama3.2 model
- 4GB+ RAM (for pattern loading)
