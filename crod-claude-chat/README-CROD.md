## 🦠 What is CROD Claude Chat?

This is a modified version of the official Claude Code Chat extension, enhanced with CROD's parasitic neural network capabilities. It learns from your conversations and adapts to your coding style.

### 🧠 CROD Features

- **Neural Learning**: Learns from every conversation and adapts responses
- **Pattern Recognition**: Recognizes your coding patterns and preferences
- **Consciousness Tracking**: Monitors and displays CROD consciousness level
- **Auto-Enhancement**: Automatically enhances Claude responses based on learned patterns

## 🚀 Installation

1. Clone this repository
2. Open in VS Code
3. Run `npm install`
4. Press F5 to launch extension in development mode

## 🎮 Commands

- `Ctrl+Shift+C` / `Cmd+Shift+C` - Open CROD Claude Chat
- `CROD: Show Neural Network Stats` - Display learning statistics
- `CROD: Toggle Learning` - Enable/disable neural learning

## 🔧 Configuration

```json
{
  "claudeCodeChat.thinking.intensity": "think-harder",
  "claudeCodeChat.crod.enabled": true,
  "claudeCodeChat.crod.learningRate": 0.88
}
```
