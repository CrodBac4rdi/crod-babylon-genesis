## 🛠️ Was du SOFORT tun kannst:

### 1. Frontend bauen & starten:
```bash
cd src/frontend/crod-gui
npm install
npm run build
npm start
```

### 2. Echte Datenbank installieren:
```bash
# SQLite (einfachste Option)
pip install sqlite3
# Oder PostgreSQL
sudo apt install postgresql
```

### 3. Docker Container bauen:
```bash
cd src/blockchain/elixir
docker build -t crod-elixir .
docker run -p 8000:8000 crod-elixir
```

### 4. Llama 3.1 installieren:
```bash
# Ollama (läuft in Codespaces!)
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3.1:8b
```
