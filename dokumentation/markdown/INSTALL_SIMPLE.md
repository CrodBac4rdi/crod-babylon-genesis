# 🦠 CROD PARASIT - Einfache Installation

**Für absolute Anfänger** - Diese Anleitung führt dich Schritt für Schritt durch die Installation.

## 📋 Was du brauchst

- **Computer** mit Linux (Ubuntu/Debian) oder macOS
- **Internetverbindung** (für Downloads)
- **10GB freier Speicherplatz**
- **30 Minuten Zeit**

## 🚀 Schritt-für-Schritt Installation

### 1. Terminal öffnen
- **Linux**: Drücke `Ctrl + Alt + T`
- **macOS**: Drücke `Cmd + Space`, tippe "Terminal" und drücke Enter

### 2. Repository herunterladen
```bash
# Kopiere diese Zeile und drücke Enter
git clone https://github.com/your-username/crod-babylon-genesis.git
```

### 3. In den Ordner wechseln
```bash
# Kopiere diese Zeile und drücke Enter
cd crod-babylon-genesis
```

### 4. Installer ausführen
```bash
# Kopiere diese Zeile und drücke Enter
chmod +x ULTIMATE_INSTALL.sh && ./ULTIMATE_INSTALL.sh
```

**⏱️ Jetzt warten!** Der Installer läuft automatisch und installiert alles (dauert 10-20 Minuten)..

### 5. CROD starten
```bash
# Kopiere diese Zeile und drücke Enter
./START_CROD.sh
```

## 🎉 Fertig!

CROD öffnet sich automatisch in einem neuen Fenster. Du siehst:
- **Chat-Interface** links - Hier kannst du mit der AI sprechen
- **File Explorer** rechts - Hier siehst du deine Dateien
- **Code Editor** in der Mitte - Hier schreibst du Code

## 🎯 Erste Schritte

1. **Tippe "Hello" in den Chat** - Die AI antwortet
2. **Erstelle eine neue Datei** - Klicke auf "New File"
3. **Schreibe etwas Code** - Probiere Python oder JavaScript aus
4. **Lass die AI helfen** - Frage sie nach Code-Beispielen

## ❓ Probleme?

### "git: command not found"
```bash
# Installiere Git
sudo apt-get install git  # Linux
brew install git           # macOS
```

### "Permission denied"
```bash
# Stelle sicher, dass das Script ausführbar ist
chmod +x ULTIMATE_INSTALL.sh
```

### "npm not found"
```bash
# Der Installer installiert npm automatisch
# Wenn es nicht funktioniert, starte das Terminal neu
```

## 💡 Tipps

- **Geduld haben** - Die erste Installation dauert etwas
- **Terminal offen lassen** - Schließe es nicht während der Installation
- **Bei Fehlern** - Kopiere die Fehlermeldung und frage die AI im Chat

## 🚨 Notfall-Kommandos

Falls etwas schief geht:

```bash
# Alles neu starten
./ULTIMATE_INSTALL.sh

# Nur die App starten
./START_CROD.sh

# Logs anzeigen
cat logs/install.log
```
