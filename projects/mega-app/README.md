## 🚨 Troubleshooting

1. **Blockchain API not connecting**
   - Make sure API is running on port 8001
   - Check browser console for errors
   - App will use mock data if API is down

2. **3D visualization not showing**
   - WebGL must be enabled in browser
   - Try refreshing the page
   - Check GPU acceleration settings

3. **Charts not updating**
   - Clear browser cache
   - Check console for JavaScript errors
