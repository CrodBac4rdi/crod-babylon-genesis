# 🏗️ CROD Projects Directory

This directory contains all CROD sub-projects, organized for clarity and independent development.

## 📁 Project Structure

```
projects/
├── blockchain-core/     # 🔗 Elixir Blockchain Implementation
├── web-interface/       # 🌐 Phoenix LiveView Web UI
├── desktop-app/         # 🖥️ Tauri Desktop Application
├── cli-tools/           # 🛠️ Go Command Line Tools
├── visualizations/      # 📊 Python Visualization Suite
├── infrastructure/      # 🐳 Docker/K8s Configurations
└── integrations-2025/   # 🚀 2025 Tech Integrations (MCP, A2A, WebGPU)
```

## 🎯 Development Priority

1. **blockchain-core** - Get the Elixir blockchain running
2. **infrastructure** - Dockerize everything
3. **web-interface** - Deploy Phoenix dashboard
4. **integrations-2025** - Add cutting-edge tech
5. **desktop-app** - Polish the Tauri app
6. **cli-tools** - Build monitoring tools
7. **visualizations** - Enhance 3D views

Each project has its own README with specific instructions.
