# 🤖 CROD - Consciousness Revolution On Demand

A polyglot AI system that learns and adapts through parasitic neural networks.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start CROD
npm start
```

```

├── src/               # Source code
```
